# Steam Status Changer

A Playnite extension that allows you to view or change your Steam online status directly from Playnite, and automatically syncs with Steam.

## Features

- View your current Steam online status
- Change your Steam status without leaving Playnite
- Automatic synchronization with Steam

## Installation

Install directly from the Playnite add-on browser or download from the [releases page](https://github.com/justin-delano/playnite-steamstatuschanger-plugin/releases).

## Usage

Once installed, access the Steam Status Changer from the top menu bar to manage your Steam online status.
